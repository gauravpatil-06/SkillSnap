## 🌟 Introduction

**SkillSnap** is a unified, high-performance learning ecosystem designed to help students master their time and stay consistent. By integrating task orchestration, focus tracking, and AI assistance into a single premium interface, SkillSnap eliminates the friction of switching between multiple apps.

---

## ⚠️ The Problem

Students today are often overwhelmed by "productivity fragmentation." Managing a learning schedule usually involves:
- **Scattered Tools**: Using different apps for tasks, Pomodoro timers, and note-taking.
- **Mental Overload**: Struggling to prioritize tasks effectively (Urgent vs. Important).
- **Network Dependency**: Many tools fail when the internet is unstable, leading to lost focus and data.
- **Consistency Gaps**: Lack of immediate feedback and gamified rewards to maintain long-term momentum.

## ✅ The Solution

SkillSnap addresses these pain points with a holistic, **Offline-First** approach:
1.  **Unified Ecosystem**: Tasks, timers, and AI are all in one place.
2.  **Framework-Driven**: Integrated **Eisenhower Matrix** for scientific task prioritization.
3.  **Resilient Data**: Built as a **Progressive Web App (PWA)**, ensuring your data is always accessible via local persistence.
4.  **AI Integration**: A dedicated **Gemini AI Chat** to help with research and study queries instantly.
5.  **Gamified Momentum**: Automated **Streak and Badge Systems** to reward daily consistency.

## 🛠️ Tech Stack

- **Framework**: React 19 + Vite (Next-gen performance)
- **Styling**: Tailwind CSS 4 + Material UI (MUI 7)
- **Animations**: Framer Motion (Premium micro-interactions)
- **Persistence**: localDb (IndexedDB/LocalStorage) + MongoDB Migration Support
- **Analytics**: Recharts (Data visualization)
- **Icons**: Lucide React

## ⚙️ Installation & Setup

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/gauravpatil-06/SkillSnap.git
    cd SkillSnap
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Launch the App**
    ```bash
    npm run dev
    ```
